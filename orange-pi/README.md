# Orange Pi Linux Lab

## Overview

This lab documents Linux installation and compatibility testing on Orange Pi hardware. It demonstrates the ability to work across single-board-computer platforms, evaluate image support, validate hardware, troubleshoot device-tree or driver limitations, and document differences without assuming Raspberry Pi procedures apply unchanged.

## Learning Objectives

- Identify the exact Orange Pi model and revision.
- Select a trustworthy operating-system image.
- Verify image checksums when provided.
- Validate boot, display, network, USB, storage, audio, and thermals.
- Document unsupported or partially supported features.
- Apply updates cautiously.
- Compare vendor and community images.
- Maintain a recovery image.

## Key Principle

Orange Pi models can differ significantly. Board name, processor, memory, storage options, Wi-Fi chipset, and image compatibility must be confirmed for the exact device.
